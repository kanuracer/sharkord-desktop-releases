# Sharkord Desktop Releases

Latest public release only.

## v0.3.99

- Fixes the channel settings modal layout so the private-channel checkbox stays inline and compact instead of stretching like a slider.
- Tightens channel settings modal sizing to remove excess vertical whitespace and keep footer/status layout clean.
- Treats missing `plugins.getSlots` as optional compatibility behavior so older Sharkord servers no longer show a plugin-slots load error.
