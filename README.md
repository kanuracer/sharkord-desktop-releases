# Sharkord Desktop Releases

Latest public release only.

## v0.3.100

- Fixes Channel Settings permissions compatibility for Sharkord servers that do not expose `channels.getPermissions` yet.
- Polishes Channel Settings permissions layout: stable scroll gutters, safer footer/status wrapping, clearer active permission states, and less clipping around the reload control.
- Fixes the profile edit popup preview so banner color/image and multiline bio display cleanly.

## Downloads

- macOS Apple Silicon: `sharkord-desktop-0.3.100-darwin-arm64.zip`
- Linux x64: `sharkord-desktop-0.3.100-linux-amd64.tar.gz`
- Windows x64: `sharkord-desktop-0.3.100-windows-amd64.exe`
