# Sharkord Desktop Releases

Latest public release only.

## v0.3.98

- Adds thread/reply parity: reply targets, thread panel, parent previews, thread compose, and typing signals for replies.
- Adds richer composer behavior: formatting toolbar, content-editable input, plugin command affordances, dynamic compose handling, and Arrow-Up edit shortcut.
- Adds channel permission overrides in channel settings for roles/users with safe public-channel warning.
- Adds native plugin UI slots for topbar, chat actions, connect, and content-wrapper surfaces.
