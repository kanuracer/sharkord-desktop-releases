# Sharkord Desktop Releases

Latest public beta release.

## v0.4.2 Beta

- Replies from mentions/notifications target the correct chat.
- Voice member drag & drop between channels is more robust.
- Text-channel unread counters no longer increase by 2×.
- Stable channel stays on the approved stable version.
