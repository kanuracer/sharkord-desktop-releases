# Sharkord Desktop Releases

Latest public beta release.

## v0.4.6 Beta

- Message deletes now update the desktop UI immediately.
- Delete buttons now respect message ownership and manage-message permission.
- Remote deletes from web or another client now clear chats, DMs, cached views, and threads.
- Stable channel stays on the approved stable version.

## Downloads

- macOS Apple Silicon: `sharkord-desktop-0.4.6-darwin-arm64.zip`
- Linux x64: `sharkord-desktop-0.4.6-linux-amd64.tar.gz`
- Windows x64: `sharkord-desktop-0.4.6-windows-amd64.exe`

Verify with `SHA256SUMS.txt`.
