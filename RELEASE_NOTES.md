# Sharkord Desktop v0.4.3 Beta

## Changes

- Central Direct Messages hub is now available from the Sharkord home/logo entry.
- Direct Messages now support server-scoped sending, unread/read state, and file upload rules.
- User profile popovers now include Direct Message, member-since, and manage-user actions.
- Admin user details now show activity, message/link/file history, storage usage, and quota context.
- Voice builds now include an Opus codec guard so Desktop/Web audio negotiation stays on `audio/opus`.
- Stable channel stays unchanged; this build is published to the Beta update channel.
